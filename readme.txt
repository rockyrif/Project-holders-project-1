This project is about under 5$ cost simple form for scholarship application system using google sheet as database.

Link to website : https://rockyrif.github.io/Project-holders-project-1/1%20login-page/
Link to database (Google sheet link) : https://docs.google.com/spreadsheets/d/12XrHcDPgIj3LuqLGNssO3jheNM9VpoesRZJad88CLhY/edit?usp=sharing

How to use:

1. go to https://rockyrif.github.io/Project-holders-project-1/1%20login-page/ and signup
2. in the next page click "apply here" 
3. next you can see a form fill and submit.
4. go to https://docs.google.com/spreadsheets/d/12XrHcDPgIj3LuqLGNssO3jheNM9VpoesRZJad88CLhY/edit?usp=sharing and check the data that you submit
